Welcome to Kytos SDN Plaforms's nightly docker image
====================================================

This is a docker image with Kytos SDN Plataform already installed ready to run.

You can easily run this image with specific napps installed running the command
below.

$ docker run -e KYTOS_NAPPS='kytos/of_core,kytos/of_lldp' -it kytos/nightly


If you want run the docker in background you can use the command below.

$ docker run -e KYTOS_NAPPS='kytos/of_core,kytos/of_lldp' -dt kytos/nightly


For more information please visit: https://kytos.io/
